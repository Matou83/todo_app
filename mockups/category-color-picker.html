Lors de la création d'une catégorie, une palette de 10 couleurs s'affiche sous le champ texte. L'utilisateur clique sur une couleur pour la sélectionner — l'aperçu du chip se met à jour en temps réel.

✓ Simple, rapide, cohérent visuellement
✓ Pas de risque de couleurs illisibles
✓ Aperçu instantané du chip
✗ Choix limité à 10 couleurs

🖥 Ouvre le sélecteur de couleur natif du système (macOS / Windows / Android)

Un <input type="color"> s'affiche sous le champ texte. Clic → ouvre le color picker natif du système d'exploitation, avec accès à toutes les couleurs et aux couleurs récentes.

✓ Liberté totale de couleur
✓ Natif = zéro code à écrire
✓ Couleurs récentes mémorisées
✗ UI du picker différente sur chaque OS
✗ Risque de couleurs illisibles (trop claires)

Les swatches s'affichent directement sous le champ de création, compactes et accessibles. Un dot de couleur à gauche de l'input indique la couleur sélectionnée. L'aperçu du chip est visible en temps réel.

✓ Compact, tout visible en une ligne
✓ Aperçu instantané dans la filterbar
✓ UX mobile-friendly
✗ Choix limité (mais même palette qu'Option A)
